# resource-index-ssl

## 项目简介

本项目是一个独立的 HTML 页面归档仓库，用于收集、整理与 SSL/TLS 相关的资源索引页面。仓库内所有页面均为静态 HTML 文件，可直接通过浏览器打开查看，不依赖后端服务或特定运行环境。

这些页面内容主要围绕 SSL 证书、HTTPS 部署、加密协议等主题，提供结构化的信息索引与导航。本仓库不针对任何特定域名或网站，仅作为通用资源归档使用。

## 目录结构

```
resource-index-ssl/
├── index.html          # 主索引页面，可快速浏览所有归档资源
├── pages/              # 按主题分类的独立 HTML 页面
│   ├── ssl-basics.html
│   ├── tools.html
│   └── references.html
├── assets/             # 页面引用的静态资源（CSS、图片等）
└── README.md           # 本文件
```

- `index.html`：仓库入口页面，列出所有归档页面的链接与简要说明。
- `pages/`：存放各主题的独立 HTML 文件，文件名反映内容类别。
- `assets/`：页面样式与图片等资源文件，确保页面正常渲染。

## 页面归档说明

本仓库收录的 HTML 页面均为静态资源归档，内容可能包含：

- SSL/TLS 协议相关概念与术语说明
- 常用工具或命令行操作的参考索引
- 部署 HTTPS 时可能涉及的技术要点汇总
- 其他与 SSL 相关的资料整理

所有页面均以信息性、参考性为目的，不包含任何推广、营销或诱导点击内容。页面内容可能随技术发展更新，但不会针对具体网站或服务进行优化。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 直接用浏览器打开 `index.html` 即可浏览所有归档页面。
3. 也可以直接打开 `pages/` 下的独立 HTML 文件查看特定主题。

无需安装任何软件或依赖，所有页面均为纯前端 HTML。
